## Image Optimization

All images should be:
- Optimized for web (compressed)
- In appropriate formats (JPEG for photos, PNG for graphics)
- Responsive (multiple sizes if needed)
- With proper alt text

## Instagram Integration

To get images from Instagram:
1. Use Instagram Basic Display API
2. Download high-resolution images
3. Optimize for web use
4. Update image paths in `config/site-config.json`

## Placeholder Images

If you don't have the actual images yet, you can use placeholder services:
- https://picsum.photos/ for random images
