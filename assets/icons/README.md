## Icon Guidelines

### Design Requirements
- Use the same color scheme as the website (#d4a574, #c49660)
- Maintain consistent style across all icons
- Ensure icons are recognizable at small sizes
- Use simple, clean designs

### Technical Requirements
- PNG format for most icons
- ICO format for favicon
- Transparent backgrounds where appropriate
- Optimized file sizes
- High resolution for retina displays

### Logo Design
The main logo should:
- Represent the bakery brand
- Work well in both light and dark contexts
- Be scalable from 16px to 512px
- Include the brand name or initials
- Use bakery-related imagery (bread, wheat, etc.)

## Instagram Integration

For the Instagram logo:
1. Download the official Instagram logo
2. Adapt it to match your brand colors
3. Ensure it's recognizable as Instagram
4. Test at different sizes

## File Naming Convention

Use descriptive names:
- `icon-[size].png` for PWA icons
- `[function]-[size].png` for specific icons
- `[brand]-logo.png` for logos
- `favicon.ico` for favicon

## Testing

Test icons on:
- Different devices and screen sizes
- Various browsers
- Light and dark themes
- Different backgrounds
